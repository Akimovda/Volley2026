# Volley — Project Status

## Реализовано
- Laravel 12 + Jetstream
- PostgreSQL
- Telegram Login (primary)
- Привязка Telegram к аккаунту
- VK ID Login (OAuth 2.1 + PKCE)
- Защита от дублей email / external IDs
- Редирект после логина → `/events`
- Базовые поля профиля
- UI завершения профиля (`/profile/complete`)
- Анкета игрока (персональные данные, уровни)
- Автозапись на мероприятие после заполнения профиля
- Flash-уведомления о статусе записи
- Страница мероприятий `/events`
- Запись на мероприятие
- Отмена записи на мероприятие
- Статусы мероприятий: «Записаться / Уже записан / Отменить запись»
- Централизованные стили проекта (`resources/css/volley.css`)

## В процессе
- Улучшение UX профиля (скролл, подсветка, тексты)
- Уточнение требований мероприятий (команды, лимиты)
