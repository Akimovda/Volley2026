# Volley — Project Status

## Текущая версия
**v0.1-alpha** (в Git теги могут быть semver: `vX.Y.Z[-alpha.N]`)

## Реализовано
- Laravel 12 + Jetstream
- PostgreSQL
- Telegram Login (primary)
- VK ID Login (secondary)
- Защита от дублей email
- Redirect после логина → `/events`
- Страница мероприятий `/events` (список)
- Запись/отмена записи на мероприятие (`event_registrations`)
- Проверка заполненности профиля **при записи на мероприятие**, а не при логине
- `/profile/complete` поддерживает `required=` и legacy `section=`
- Анкета игрока `/profile/extra`:
  - дата рождения
  - уровни классика/пляж (в базе числа, в UI названия)
  - возрастные ограничения уровней (<18 — 3 уровня)
  - lock: выбор уровня пользователем 1 раз (повторно меняют админ/организатор)
  - амплуа классики (несколько + основное)
  - зоны пляжа (2/4 + основная) или универсал
  - автозапись на мероприятие после заполнения профиля
- [x] Публичные страницы /users и /user/{id}
- [x] Исправлен navigation-menu для гостей (не падает на Auth::user())
- [x] Добавлены поля профиля: gender, height_cm
- [x] Правила видимости sensitive полей через Gate view-sensitive-profile
