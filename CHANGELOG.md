# Changelog

All notable changes to this project will be documented in this file.

## v0.1-alpha

### Features
- Telegram + VK ID авторизация (primary/secondary)
- Страница мероприятий `/events`
- Запись/отмена записи на мероприятие
- Проверка профиля при записи (не при логине)
- `/profile/complete` с `required=` и поддержкой legacy `section=`
- Анкета игрока `/profile/extra` + подсветка required + автозапись
- Централизованные стили проекта (`resources/css/volley.css`)

### Documentation
- Добавлены docs: status, auth, events, profile, dev, architecture, ADR
